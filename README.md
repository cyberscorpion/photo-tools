# Photo Tools

A browser-based photo editor built with React, Vite, and Fabric.js.

**Live demo:** https://cyberscorpion.github.io/photo-tools

## Features

**Drawing & Editing Tools**
- Select / Move, Hand / Pan, Zoom
- Crop, Brush, Eraser, Lasso
- Text, Rectangle, Ellipse
- Eyedropper

**Adjustments**
- Brightness, Contrast, Saturation, Hue
- Exposure, Warmth, Tint
- Shadows, Highlights, Sharpness, Blur

**Filters**
- Grayscale, Sepia, Invert, Vintage, Vignette

**Other**
- Layers panel
- History (undo/redo)
- Histogram
- Contour detection
- Export as PNG, JPEG, or WebP with quality control
- Copy to clipboard
- Keyboard shortcuts

## Tech Stack

| Package | Role |
|---|---|
| React 18 | UI framework |
| Vite 5 | Build tool |
| Fabric.js 6 | Canvas engine |
| Zustand | State management |
| Lucide React | Icons |

## Getting Started

```bash
npm install
npm run dev
```

Open http://localhost:5173 in your browser.

## Scripts

| Command | Description |
|---|---|
| `npm run dev` | Start dev server |
| `npm run build` | Production build |
| `npm run preview` | Preview production build locally |
| `npm run deploy` | Build and deploy to GitHub Pages |
